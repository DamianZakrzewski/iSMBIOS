# DmiProperty.Processor.CoreEnabled2 property

Gets a value representing the key to retrieve the property value.

Number of enabled cores per processor socket. Supports core enabled counts &gt;255 If this field is present, it holds the core enabled count for the processor socket. Core Enabled will also hold the core enabled count, except for core counts that are 256 or greater. In that case, core enabled shall be set to FFh and core enabled 2 will hold the count.
