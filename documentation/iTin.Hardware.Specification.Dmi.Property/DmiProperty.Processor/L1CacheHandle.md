# DmiProperty.Processor.L1CacheHandle property

Gets a value representing the key to retrieve the property value.

Handle of a cache information structure that defines the attributes of the primary (Level 1) cache for this processor.

* For version 2.1 and version 2.2 implementations, the value is 0FFFFh if the processor has no L1 cache.
* For version 2.3 and later implementations, the value is 0FFFFh if the Cache Information structure is not provided.
