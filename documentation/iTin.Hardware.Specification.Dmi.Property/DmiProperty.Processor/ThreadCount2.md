# DmiProperty.Processor.ThreadCount2 property

Gets a value representing the key to retrieve the property value.

Number of threads per processor socket. Supports thread counts &gt;255 If this field is present, it holds the thread count for the processor socket. Thread Count will also hold the thread count, except for thread counts that are 256 or greater. In that case, thread count shall be set to FFh and thread count 2 will hold the count.
